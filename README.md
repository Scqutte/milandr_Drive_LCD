# Milandr Drive LCD Game

Проект представляет собой простую игру для микроконтроллера Миландр 1986ВЕ93У.

Идея проекта: на LCD-дисплее отображается самолётик, которым пользователь управляет с помощью джойстика. Также в проекте планируется вывод звуковых эффектов.

## Основные возможности

- вывод графики на LCD-дисплей;
- отображение самолёта на экране;
- управление самолётом с помощью джойстика;
- обработка пользовательского ввода;
- вывод звука;
- сборка проекта через GNU Make;
- использование Arm GNU Toolchain.
